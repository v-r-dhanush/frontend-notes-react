*string and its methods

string : It is collection of character.
        -In string every character has index value and it start from zero.
        -In string blank space is also consider as string.

-There are three method which index of searching string

1.indexof(searchingString,startIndex?) : -It will search string from left to right and return index of first occurences of string
                                         -It searching string is not found then return -1 as result.
                                         Eg:
                                         document.write(`msg.indexOf("all"):${msg.indexOf("all")}<br>`);
                                         document.write(`msg.indexOf("all",16):${msg.indexOf("all",16)}<br>`);

2.lastIndexof(searchingString,startIndex?) : -It will search string from right to left and return index of first occurences of string
                                             -It searching string is not found then return -1 as result.
                                             Eg:
                                             document.write(`msg.lastIndexOf("you"):${msg.lastIndexOf("you")}<br>`);
                                             document.write(`msg.lastIndexOf("you",28):${msg.lastIndexOf("you",28)}<br>`);

3.search(searchingString) : -It will search string from left to right and return index of first occurences of string
                            -It searching string is not found then return -1 as result.
                            -It is faster than indexof method.
                            Eg:
                            document.write(`:msg.search("changepond")${msg.search("changepond")}<br>`);

-There are three method which retun part of searching string

1.slice(startIndex,endIndex?) : 
-It will return part of string from big string.
-If we write +ve index then will search index from left to right.
-If we write -ve index then will search index from right to left.
Eg:
document.write(`msg.slice(15):${msg.slice(15)}<br>`);
document.write(`msg.slice(15,30):${msg.slice(15, 32)}<br>`);
document.write(`msg.slice(-20):${msg.slice(-20)}<br>`);
document.write(`msg.slice(-20,-4):${msg.slice(-20, -4)}<br>`);

2.substring(startIndex,endIndex?) : 
-It will return part of string from big string.
-If we write +ve index then will search index from left to right.
Eg:
document.write(`msg.substring(10):${msg.substring(10)}<br>`);
document.write(`msg.substring(10,25):${msg.substring(10, 25)}<br>`);

3.substr(startIndex,totalNumberOfCharacter) : 
-It will return part of string from big string.
-If we write +ve index then will search index from left to right.
Eg:
document.write(`msg.substr(5,10):${msg.substr(5, 10)}<br>`);

-charAt(index) : It return character from index number.
-charCodeAt() : It return ASCII code.
Eg:
document.write(`msg.charAt(0):${msg.charAt(0)}<br>`);
document.write(`msg.charCodeAt(0):${msg.charCodeAt(0)}<br>`);
